# @dhrv/prettier-config

> dhrv's preferred [prettier](https://prettier.io) config

## Usage

**Install**:

```bash
$ yarn add --dev @0xdhrv/prettier-config
```

```bash
$ npm install --save-dev @0xdhrv/prettier-config
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "prettier": "@0xdhrv/prettier-config"
}
```
